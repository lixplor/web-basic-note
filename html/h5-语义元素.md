# HTML 5 语义元素

* 语义=意义, 语义元素=元素的意义
* 语义元素能够清楚描述其意义给浏览器和开发者


## H5中新的语义元素

* `<header>`: 描述文档的头部区域
* `<nav>`: 定义导航链接部分
* `<section>`: 定义文档中的节
* `<article>`: 定义独立的内容
* `<aside>`: 定义页面珠区域内容之外的内容, 如侧边栏
* `<figcaption>`: 定义figure元素的标题
* `<figure>`: 定义独立的流内容
* `<footer>`: 描述文档底部区域

```shell
+-----------------------+
|     <header>          |
+-----------------------+
|      <nav>            |
+------------+----------+
|  <section> |          |
+------------+ <aside>  |
|  <article> |          |
+------------+----------+
|      <footer>         |
+-----------------------+
```
