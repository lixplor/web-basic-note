# HTML简介

* `HTML`是`HyperText Markup Language`的缩写, 超文本标记语言
* html文档后缀有以下两种, 没有区别
    - `.html`
    - `.htm`

## html结构

```html
<!DOCTYPE html>                // 声明为HTML5
<html>                         // html页面根元素
<head>                         // 包含文档头部信息, 元数据等
  <meta charset="utf-8">
  <title>标题</title>          // 文档的标题
</head>
<body>                         // 包含可见的页面内容
  <h1>标题</h1>                
  <p>段落</p>
</body>
</html>
```
