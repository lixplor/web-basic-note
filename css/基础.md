# CSS基础


## CSS文件的使用

3种插入样式表的方法:
* 外部样式表 
    - `<link>`标签
    - `.css`文件名后缀
* 内部样式表: 
    - `<style></style>`标签
* 内联样式
    - `style`属性

样式优先级: 浏览器默认 < 外部样式表 < 内部样式表 < 内联样式
样式的继承: 多种方式定义的样式, 未被覆盖的样式会被继承并拥有

```html
<!-- 外部样式表 -->
<head>
  <link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

<!-- 内部样式表 -->
<head>
  <style>
    hr {color:sienna;}
    p {margin-left:20px;}
    body {background-image:url("images/back40.gif");}
  </style>
</head>

<!-- 内联样式 -->
<p style="color:sienna;margin-left:20px">This is a paragraph.</p>
```


## 语法

### 格式

* `属性:值`为声明
* 属性和值用`:`冒号分开
* 声明以`{}`大括号包裹

```css
选择器 {属性:值;属性:值;...}

p {
  color:red;
  text-align:center;
}
```

### 注释

```css
/*注释可以一行*/
/*注释也
  可以
  多行*/
```

### 选择器和选择符

* 选择器
    - id选择器
        - 为指定id的html元素指定样式
        - 格式: `#id {}`
        - id不能以数字开头, 数字开头的id在FireFox不起作用
    - class选择器
        - 为指定class的html元素指定样式
        - 格式: `.class {}`
        - class不能以数字开头, 数字开头的class在FireFox不起作用
    - 元素也可作为选择器
        - 为指定元素指定样式
        - 格式: `h1 {}`
    - 元素和类选择器
        - 为指定class的指定元素指定样式
        - 格式: `h1.class {}`
    - 分组选择器
        - 多个同类选择器可以同时进行设置, 用`,`区分
        - 格式: `p, h1, h2 {}`
    - 嵌套选择器
        - 选择器内部的选择器
        - 格式: `p {} .marked {} .content {}`
    - 后代选取器
        - 为指定元素的所有后代元素设置样式, 用` `空格隔开
        - 格式: `div div p {}`
    - 子元素选择器
        - 仅为指定元素的子元素设置样式, 用`>`大于号隔开
        - 格式: `div>p {}`
    - 相邻兄弟选择器
        - 选择紧接在另一元素后的元素, 且两者有相同父元素的元素设置样式, 用`+`加号分隔
        - 格式: `div+p {}`
    - 普通相邻兄弟选择器
        - 选取所有指定元素的相邻兄弟元素指定样式, 用`-`减号分隔
        - 格式: `div-p {}`
    - 属性选择器
        - 为包含指定html属性的元素设置样式, 用`[属性]`中括号包裹属性
        - 格式: `[href] {}`
* 伪类
伪类用于添加一些选择器的特殊效果
    - 语法
        - `选择器:伪类 {}`
        - `选择器.类:伪类 {}`
    - 伪类
        - `:checked`: 所有选中的选择器元素
        - `:disabled`: 所有禁用的选择器元素
        - `:empty`: 所有没有子元素的选择器元素
        - `:enabled`: 所有启用的选择器元素
        - `:first-of-type`: 每个父元素是选择器元素的第一个指定子元素
        - `:in-range`: 选择器元素指定范围内的值
        - `:invalid`: 所有无效的选择器元素
        - `:last-child`: 所有选择器元素的最后一个子元素
        - `:last-of-type`:每个选择器元素是其父元素的最后一个指定元素
        - `:not(选择器)`: 所有选择器元素以外的元素
        - `:nth-child(n)`: 所有选择器元素的第n个子元素
        - `:nth-last-child(n)`: 所有选择器元素的倒数第n个子元素
        - `:nth-last-of-type(n)`: 所有选择器倒数第n个位选择器元素的子元素
        - `:nth-of-type(n)`: 所有选择器元素第二个为选择器元素的子元素
        - `:only-of-type`: 所有仅有一个子元素为选择器元素的元素
        - `:only-child`: 所有仅有一个子元素的选择器元素
        - `:optional`: 没有`required`的元素属性
        - `:out-of-range`: 指定范围以外的值的元素属性
        - `:read-only`: 只读属性的元素属性
        - `:read-write`: 没有只读属性的元素属性
        - `:required`: 有`required`属性指定的元素属性
        - `:root`: 文档根元素
        - `:target`: 当前活动锚点元素(`#news:target`)
        - `:valid`: 所有有效值的属性
        - `:link`: 所有未访问链接
        - `:visited`: 所有已访问链接
        - `:active`: 正被点击的链接
        - `:hover`: 鼠标悬浮链接上的状态
        - `:focus`: 元素输入后具有焦点
        - `:first-letter`: 每个选择器元素的第一个字母
        - `:first-line`: 每个选择器元素的第一行
        - `:first-child`: 选择器匹配属于任意元素的第一个子元素的选择器元素
        - `:before`: 在每个选择器元素之前插入内容
        - `:after`: 在每个选择器元素之后插入内容
        - `:lang(语言)`: 为选择器元素的lang属性选择一个初始值
* 伪元素
伪元素用于添加选择器的特殊效果
    - 语法
        - `选择器:伪元素 {}`
        - `选择器.类:伪元素 {}`
    - 伪元素
        - 同伪类


```css
/* id选择器 */
#para1 {
  text-align:center;
  color:red;
}

/* class选择器 */
.center {
  text-align:center;
}

/* 元素作为选择器 */
p {
  color:red;
}

/* 元素和class组合 */
p.red {
  color:red;
}

a.one:link {
  color:#ff0000;
}

/* 多个同类选择器 */
table, th, td {
  border:1px solid black;
}

/* 后代选择器 */
div p {
  background-color: yellow;
}

/* 子元素选择器 */
div>p {
  color:red;
}

/* 相邻兄弟选择器 */
div+p {
  color:blue;
}

/* 普通相邻兄弟选择器 */
div-p {
  color:red;
}

/* 属性选择器 */
[href] {
  color:red;
}

[href="http://www.baidu.com"] {
  color:blue;
}
```

## CSS的定位机制

* 3种定位机制:
    1. 标准文档流: 从左到右, 从上到下
    2. 浮动: 左, 右. 浮动会将盒子脱离文档流, 其他盒子无视这个元素, 但其他盒子内的文本依然会为浮动元素让出位置
    3. 绝对定位: 相对于浏览器窗口定位. 会将盒子脱离文档流, 其他盒子及其文本都会无视这个元素


## 样式

* 颜色
    - 十六进制: `#ff00ff`
    - RGB: `rgb(255, 0, 123)`
    - 颜色名称: `red`
* 长度
    - 相对长度
        - `em`: 相对于当前元素字体尺寸的倍数. 一般浏览器字体默认16px, 则2em == 32px
        - `ex`: 相对于英文字母小写x的倍数
        - `ch`: 数字0的宽度
        - `rem`: 根元素(`<html>`)的font-size
        - `vw`: viewpoint width, 视窗宽度, 1vw == 视窗宽度的1%
        - `vh`: viewpoint height, 视窗高度, 1vh == 视窗高度的1%
        - `vmin`: vw和vh中较小的值
        - `vmax`: vw和vh中较大的值
        - 百分比
    - 绝对长度
        - `cm`: 厘米
        - `mm`: 毫米
        - `in`: 英寸(1in == 96px == 2.54cm)
        - `px`: 像素(1px == 1/96in)
        - `pt`: point(1pt 约等于 1/72in)
        - `em`: pica, (1pc = 12pt)
* 背景
定义html元素的背景
    - `background-color`: 元素背景颜色
    - `background-image`: 元素背景图像, 默认平铺重复显示
        - `url('图片地址')`
    - `background-repeat`: 背景图像平铺方向
        - `repeat-x`: 水平平铺
        - `repeat-y`: 垂直平铺
        - `no-repeat`: 不平铺
    - `background-attachment`: 背景图像是否固定或者随着页面的其余部分滚动
        - `scroll`: 背景图片随页面的其余部分滚动, 默认值
        - `fixed`: 固定
        - `inherit`: 从父元素继承
    - `background-position`: 设置图像在背景中的位置
        - `left`
        - `top`
        - `right`
        - `bottom` 
    - `background`: 背景简写属性, 可以将上述属性值写在一起, 用空格分隔
        - 顺序为`background-color background-image background-repeat background-attachment background-position`, 属性无需全部使用
* 文本格式
    - `color`: 设置文本颜色
    - `direction`: 设置文本方向
        - `ltr`: 从左到右
        - `rtl`: 从右到左
        - `inherit`: 继承父元素
    - `letter-spacing`: 设置字符间距
    - `line-height`: 设置行高
    - `text-align`: 对齐元素中的文本
        - `left`: 默认, 左对齐
        - `right`: 右对齐
        - `center`: 居中对齐
        - `justify`: 两端对齐
        - `inherit`: 继承父元素
    - `text-decoration`: 向文本添加修饰
        - `none`: 默认, 标准文本
        - `underline`: 下划线
        - `overline`: 上划线
        - `line-through`: 删除线
        - `blink`: 文字闪烁(大部分浏览器无效)
        - `inherit`: 继承父元素
    - `text-indent`: 缩进元素中文本的首行
    - `text-shadow`: 设置文本阴影
        - `h-shadow`参数: 必须, 水平阴影位置, 允许负值
        - `v-shadow`参数: 必须, 垂直阴影位置, 允许负值
        - `blur`参数: 可选, 模糊距离
        - `color`: 可选, 阴影颜色
    - `text-transform`: 控制元素中的字母
        - `none`: 默认, 标准文本
        - `capitalize`: 文本中每个单词以大写字母开头
        - `uppercase`: 全部大写
        - `lowercase`: 全部小写
        - `inherit`: 继承父元素
    - `unicode-bidi`: 设置或返回文本是否被重写
        - `normal`: 默认, 不适用附加嵌入层面
        - `embed`: 创建一个附加的嵌入层面
        - `bidi-override`: 创建一个附加的嵌入层面, 重新排序取决于direction属性
        - `initial`: 设置该属性为它的默认值
        - `inherit`: 继承父元素
    - `vertical-align`: 设置元素的垂直对齐
        - `baseline`: 默认, 元素放置在父元素基线上
        - `sub`: 垂直对齐文本的下标
        - `super`: 垂直对齐文本的上标
        - `top`: 把元素的顶端与行中最高元素的顶端对齐
        - `text-top`: 把元素的顶端与父元素字体的顶端对齐
        - `middle`: 把此元素放置在父元素的中部
        - `bottom`: 把元素的顶端与行中最低的元素的顶端对齐
        - `text-bottom`: 把元素的底端与父元素字体的底端对齐
        - 长度
        - 百分比
        - `inherit`: 继承父元素
    - `white-space`: 设置元素中空白的处理方式
        - `normal`: 默认, 空白会被浏览器忽略
        - `pre`: 空白会被浏览器保留
        - `nowrap`: 文本不会换行, 直到遇到`<br>`标签为止
        - `pre-wrap`: 保留空白符序列, 但是正常地进行换行
        - `pre-line`: 合并空白符序列, 但是保留换行符
        - `inherit`: 继承父元素
    - `word-spacing`: 设置字间距
* 字体
    - `font`: 设置字体所有属性
        - `font-style`: 字体样式
        - `font-variant`: 字体异体
        - `font-weight`: 字体粗细
        - `font-size/line-height`: 字体尺寸和行高
        - `font-family`: 字体系列
        - `caption`: 标题控件使用的字体
        - `icon`: 图标标记使用的字体
        - `menu`: 下拉列表使用的字体
        - `message-box`: 对话框使用的字体
        - `small-caption`: caption字体的小型版本
        - `status-bar`: 窗口状态栏使用的字体
    - `font-family`: 指定文本的字体系列
        - `family-name/generic-family`, 每个值用逗号分隔, 可以多个, 当不满足条件时按顺序设置
    - `font-size`: 指定文本的字体大小
        - 尺寸
            - `xx-small`
            - `x-small`
            - `small`
            - `medium`
            - `large`
            - `x-large`
            - `xx-large`
        - `smaller`: 比父元素更小的尺寸
        - `larger`: 比父元素更大的尺寸
        - 长度
        - 百分比
        - `inherit`
    - `font-style`: 指定文本的字体样式
        - `normal`: 默认, 标准字体
        - `italic`: 斜体
        - `oblique`: 倾斜
        - `inherit`
    - `font-variant`: 以小型大写字体或者正常字体显示文本
        - `normal`: 默认值, 标准字体
        - `small-caps`: 显示小型大写字母
        - `inherit`
    - `font-weight`: 指定字体的粗细
        - `normal`: 默认, 标准字体
        - `bold`: 粗体字符
        - `bolder`: 更粗体
        - `lighter`: 更细体
        - `100~900`: 400等于normal, 700等于bold
        - `inherit`
* 链接
    - `a:link`: 正常, 未访问过的链接
    - `a:visited`: 已访问过的链接
    - `a:hover`: 鼠标悬浮在链接上
    - `a:active`: 链接被点击时
    - 规则: 
        - `a:hover`必须跟在`a:link`和`a:visited`后
        - `a:active`必须跟在`a:hover`后
* 列表
    - `list-style-image`: 将图像设置为列表项标志
        -`url('路径')`: 图像路径
    - `list-style-position`: 设置列表中列表项标志的位置
        - `inside`: 列表项标记防止在文本内, 环绕文本根据标记对齐
        - `outside`: 默认, 保持标记位于文本的左侧, 列表项标记放置在文本外, 环绕文本不根据标记对齐
    - `list-style-type`: 设置列表项标志的类型
        - `none`: 无标记
        - `disc`: 默认, 实心圆
        - `circle`: 空心圆
        - `square`: 实心方块
        - `decimal`: 数字
        - `decimal-leading-zero`: 0开头的数字标记(01, 02)
        - `lower-roman`: 小写罗马数字(i, ii)
        - `upper-roman`: 大写罗马数字(I, II)
        - `lower-alpha`: 小写英文字母(a, b)
        - `upper-alpha`: 大写英文字母(A, B)
        - `lower-greek`: 小写希腊字母(alpha, beta)
        - `lower-latin`: 小写拉丁字母(a, b)
        - `upper-latin`: 大写拉丁字母(A, B)
        - `hebrew`: 传统希伯来编号
        - `armenian`: 传统亚美尼亚编号
        - `georgian`: 传统乔治亚编号(an, ban)
        - `cjk-ideographic`: 简单表意数字
        - `hiragana`: 小写日文平假名(a, i)
        - `Katakana`: 大写日文平假名(A, I)
        - `hiragana-iroha`: 小写日文片假名(i, ro)
        - `katakana-iroha`: 大写日文片假名(I, RO)
    - `list-style`: 简写属性, 所有属性同时设置
* 表格
    - `border`: 表格边框
    - `border-collapse`: 表格表框是否被折叠成一个单一的边框, 或隔开
        - `collapse`: 
    - `width`: 表格宽度
    - `height`: 表格高度
* 盒子模型(Box model)
    - `margin`: 外边距
    - `border`: 边框
    - `padding`: 内边距
    - `content`: 内容
* 边框
    - `border`: 同时设置4个边的属性
    - `border-style`: 设置所有边框样式, 或分别设置样式
        - `样式`: 为4边设置相同样式
        - `样式 样式 样式 样式`: 为4边分别设置样式
    - `border-width`: 为所有边框设置宽度, 或分别设置宽度
    - `border-color`: 为所有边框的可见部分设置颜色, 或分别设置颜色
    - `border-left`: 左边框的所有属性
    - `border-top`: 上边框的所有属性
    - `border-right`: 右边框的所有属性
    - `border-bottom`: 下边框的所有属性
    - `border-left-color`: 左边框颜色
    - `border-top-color`: 上边框颜色
    - `border-right-color`: 设置右边框颜色
    - `border-bottom-color`: 下边框颜色
    - `border-left-style`: 左边框样式
        - `none`: 无边框
        - `hidden`: 无边框, 但用于表时除外
        - `dotted`: 点状
        - `dashed`: 虚线
        - `solid`: 实线
        - `double`: 双边框
        - `groove`: 双线
        - `ridge`: 三维菱形
        - `inset`: 三维凹边框
        - `outset`: 三维凸边框
    - `border-top-style`: 上边框样式
    - `border-right-style`: 右边框样式
    - `border-bottom-style`: 下边框样式
    - `border-left-width`: 左边框宽度
    - `border-top-width`: 上边框宽度
    - `border-right-width`: 右边框宽度
    - `border-bottom-width`: 下边框宽度
* 轮廓(outline)
轮廓是绘制在元素周围的一条线, 位于边框边缘的外围, 起到突出元素的作用
    - `outline`: 设置所有轮廓属性
    - `outline-color`: 轮廓颜色
        - 颜色名
        - 十六进制
        - `rgb(r,g,b)`
        - `invert`: 默认, 颜色反转
    - `outline-style`: 轮廓样式
        - 同`border-style`
    - `outline-width`: 轮廓宽度
* 外边距
    - `margin`: 设置所有外边距
        - `auto`: 结果依赖于浏览器
        - 长度
        - 百分比
    - `margin-bottom`: 下外边距
    - `margin-left`: 左外边距
    - `margin-right`: 右外边距
    - `margin-top`: 上外边距
* 内边距
    - `padding`: 设置所有内边距
        - 长度
        - 百分比
    - `padding-bottom`: 下内边距
    - `padding-left`: 左内边距
    - `padding-top`: 上内边距
    - `padding-right`: 右内边距
* 尺寸
    - `height`: 元素高度
    - `width`: 元素宽度
    - `line-heigit`: 行高
    - `max-height`: 元素最大高度
    - `max-width`: 元素最大宽度
    - `min-height`: 元素最小高度
    - `min-width`: 元素最小宽度
* 显示(display)和可见性(visibility)
    - 隐藏元素: 
        - `display:none`: 隐藏元素, 且不占用空间
        - `visibility:hidden`: 隐藏元素, 继续占用空间
    - `display`: 改变元素显示方式
        - `inline`: 作为内联元素显示
        - `block`: 作为块元素显示
* 定位
    - `position`: 将元素进行定位
        - `static`: 默认, 没有定位, 正常显示在流中, 不受top, bottom, left, right影响
        - `fixed`: 相对于浏览器窗口固定位置. 不在文档流中, 不占空间
        - `relative`: 相对于元素原本位置的定位. 在文档流中, 原本占用空间不会改变
        - `absolute`: 相对于最近的已定位的父元素, 如果没有已定位的父元素, 则相对于`<html>`. 不在文档流中, 不占空间
    - `clip`: 裁剪一个决定定位的元素
        - 形状: `rect(top, right, bottom, left)`
        - `auto`
    - `cursor`: 定义鼠标指针放在一个元素边界范围内时所用的光标形状
        - `default`: 默认光标
        - `auto`: 浏览器设置的光标
        - `url('')`: 光标地址
        - `crosshair`: 十字线
        - `pointer`: 指示链接的指针(一只手)
        - `move`: 可被移动
        - `e-resize`: 矩形框边缘可被向右移动
        - `w-resize`: 矩形框边缘可被向左移动
        - `n-resize`: 矩形框边缘可被向上移动
        - `s-resize`: 矩形框边缘可被向下移动
        - `nw-resize`: 矩形框边缘可被向左上移动
        - `ne-resize`: 矩形框边缘可被向右上移动
        - `sw-resize`: 矩形框边缘可被向左下移动
        - `se-resize`: 矩形框边缘可被向右下移动
        - `text`: 指示文本
        - `wait`: 指示程序正忙
        - `help`: 指示帮助
    - `overflow`: 当元素内容溢出其区域时的效果
        - `visible`: 默认, 内容不会被修剪, 呈现在元素框之外
        - `hidden`: 内容会被修剪, 且其余内容是不可见的
        - `auto`: 如果内容被修剪, 则浏览器会显示滚动条以便查看内容
        - `scroll`: 内容会被修剪, 但浏览器会显示滚动条以便查看内容
    - `overflow-x`: 当元素左边/右边内容溢出其区域时的效果
        - `visible`: 默认, 内容不会被修剪, 呈现在元素框之外
        - `hidden`: 内容会被修剪, 且其余内容是不可见的
        - `auto`: 如果内容被修剪, 则浏览器会显示滚动条以便查看内容
        - `scroll`: 内容会被修剪, 但浏览器会显示滚动条以便查看内容
        - `no-display`: 如果内容不适合内容框, 则删除整个框
        - `no-content`: 如果内容不适合内容框, 则隐藏整个内容
    - `overflow-y`: 当元素上边/下边内容溢出其区域时的效果
        - 同上
    - `top`: 上外边界距其包含块上边界的偏移
    - `left`: 左外边界距其包含块左边界的偏移
    - `right`: 右外边界距其包含块右边界的偏移
    - `bottom`: 下外边界距其包含块下边界的偏移
    - `z-index`: 堆叠顺序, 高值显示在低值前, 值可正可负
* 浮动
    - 浮动原理
        - 浮动会使元素向左或向右移动, 而不能上下移动
        - 元素移动直到它的外边缘碰到包含框或另一个浮动框的边框为止
        - 浮动元素前面的元素不受影响, 浮动元素后面的元素将围绕它
    - 浮动的影响
        - 浮动会导致后面的元素位置跟随在浮动元素之后, 而不是在浮动元素的下面一行
        - 浮动会导致父元素高度塌陷(边框无法撑开, 背景无法显示, margin不起作用), 因为浮动元素已经脱离了父元素
    - 浮动影响的解决办法
        - 在位置不正确的元素前添加一个空元素, 设置clear清除浮动, 这样就会阻止浮动元素的影响
    - `float`: 指定盒子元素是否可以浮动
        - `left`: 向左浮动
        - `right`: 向右浮动
        - `none`: 默认, 不浮动
    - `clear`: 清除浮动, 不允许元素周围有浮动元素
        - `left`: 左侧不允许浮动元素
        - `right`: 右侧不允许浮动元素
        - `both`: 左右两侧均不允许浮动元素
        - `none`: 默认, 允许浮动元素出现在两侧
* 水平对齐
    - 块元素对齐: 块元素独占一行, 前后都是换行符, 对齐方式如下
        - 水平居中
            - `margin-left:auto; margin-right:auto`
            - `margin:auto;`
        - 左/右对齐
            - `position:absolute; left:0px;`
            - `position:absolute; right:0px;` 
            - `float:left;`
            - `float:right;`
